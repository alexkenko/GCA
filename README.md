## About

**Georgian Crewing Agency (GCA)** is a professional crewing agency specializing in providing qualified maritime personnel to shipping companies worldwide. Our platform serves as a comprehensive solution for:

- **Crew Management**: Efficient management of maritime crew members
- **Job Listings**: Connect qualified seafarers with shipping companies
- **Registration**: Streamlined registration for both crew members and employers
- **Dashboard**: Comprehensive dashboards for managing applications and profiles

## Features

- User registration and authentication system
- Crew member and employer dashboards
- Job application and management system
- Profile management for seafarers
- Contact and inquiry system
- Responsive design for all devices

## Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher) - if using JavaScript/TypeScript
- [Python](https://www.python.org/) (v3.9 or higher) - if using Python
- [Git](https://git-scm.com/)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/alexkenko/GCA.git
cd GCA
```

2. Install dependencies:
```bash
# If using npm
npm install

# If using yarn
yarn install

# If using pip
pip install -r requirements.txt
```

3. Set up environment variables:
```bash
cp .env.example .env
# Edit .env with your configuration
```

4. Run the development server:
```bash
# For Node.js projects
npm run dev

# For Python projects
python app.py
```
